## JSON Navigator

### 개요

첨부한 JSON 파일을 파싱해 계층 구조를 나타내는 JSON Navigator 입니다.

### 기능

- `.` 점으로 수직 계층을 나타낸 json 파일의 계층도를 시각적으로 나타냅니다.
- 각 계층을 클릭 했을 때 아코디언 메뉴처럼 동작합니다.

### 실행 방법

- `npx serve -s` 명령어를 통해 실행 시킬 수 있습니다.
